# 성능 지표 추적 가이드

**프로젝트 코드**: ATCMP-2026  
**작성일**: 2025-01-27  
**버전**: 1.0

---

## 📋 개요

이 문서는 AT-CMP 프로젝트의 모든 성능 지표를 통합적으로 추적하고 관리하기 위한 가이드입니다. 개발 생산성, 제품 품질, 팀 건강도 지표를 하나의 시스템으로 통합 관리합니다.

---

## 🎯 목적

1. **통합 지표 관리**: 모든 지표를 하나의 시스템으로 관리
2. **데이터 기반 의사결정**: 지표 분석을 통한 의사결정 지원
3. **자동화**: 지표 수집 및 리포트 생성 자동화
4. **지속적 개선**: 지표 분석을 통한 지속적 개선

---

## 📊 지표 카테고리

### 1. 개발 생산성 지표

**지표**:
- Velocity (속도)
- 번다운 차트
- 스토리 포인트 추정 정확도
- 작업 완료 시간

**상세 문서**: [개발 생산성 지표](./developer-productivity-metrics.md)

---

### 2. 제품 품질 지표

**지표**:
- 가동률 (Uptime)
- 응답 시간 (Response Time)
- 에러율 (Error Rate)
- 코드 커버리지 (Code Coverage)

**상세 문서**: [제품 품질 지표](./product-quality-metrics.md)

---

### 3. 팀 건강도 지표

**지표**:
- 스프린트 완료율
- 코드 리뷰 시간
- PR 머지 시간
- 블로커 해결 시간

**상세 문서**: [팀 건강도 지표](./team-health-metrics.md)

---

## 🔄 지표 추적 프로세스

### 단계 1: 데이터 수집

#### 자동 수집

- **GitHub API**: PR, 이슈, 커밋 데이터
- **CI/CD 파이프라인**: 테스트 커버리지, 빌드 시간
- **모니터링 도구**: Sentry, Vercel Analytics

#### 수동 수집

- **스프린트 완료율**: 스프린트 종료 시 수동 입력
- **블로커 정보**: 블로커 발생 시 수동 입력

---

### 단계 2: 데이터 분석

#### 분석 주기

- **일일**: 실시간 지표 모니터링
- **주간**: 주간 리포트 생성
- **월간**: 월간 리포트 생성
- **분기별**: 분기별 종합 분석

#### 분석 항목

- 지표 값 계산
- 트렌드 분석
- 목표 달성 여부 평가
- 이상 징후 탐지

---

### 단계 3: 리포트 생성

#### 리포트 유형

1. **주간 리포트**
   - 주간 지표 요약
   - 트렌드 분석
   - 목표 달성 여부

2. **월간 리포트**
   - 월간 지표 요약
   - 트렌드 분석
   - 종합 평가

3. **분기별 리포트**
   - 분기별 지표 요약
   - 장기 트렌드 분석
   - 개선 계획 수립

---

### 단계 4: 지표 공유

#### 공유 방법

- **Slack**: 주간 리포트 자동 공유
- **Notion**: 대시보드 업데이트
- **이메일**: 월간 리포트 전송

#### 공유 대상

- 개발팀 전체
- PM 및 기술 리더
- 이해관계자 (필요시)

---

## 📈 지표 대시보드

### 대시보드 구성

#### 1. 개발 생산성 대시보드

- Velocity 트렌드
- 번다운 차트
- 추정 정확도
- 작업 완료 시간 분석

#### 2. 제품 품질 대시보드

- 가동률 통계
- 응답 시간 통계
- 에러율 통계
- 코드 커버리지

#### 3. 팀 건강도 대시보드

- 스프린트 완료율
- 코드 리뷰 시간
- PR 머지 시간
- 블로커 해결 시간

---

## 🤖 지표 자동화

### 자동화 도구

#### GitHub Actions

- 주간 리포트 자동 생성
- 지표 데이터 수집
- 리포트 배포

#### 스크립트

- 지표 데이터 수집 스크립트
- 리포트 생성 스크립트

---

### 자동화 프로세스

1. **데이터 수집** (매일)
   - GitHub API 호출
   - 모니터링 도구 데이터 수집
   - 데이터 저장

2. **리포트 생성** (주간)
   - 지표 데이터 분석
   - 리포트 생성
   - 리포트 배포

3. **알림** (목표 미달성 시)
   - 목표 미달성 알림
   - 긴급 상황 알림

## 📊 지표 목표 및 달성 평가

### 종합 목표

| 카테고리 | 지표 | 목표 | 측정 주기 |
|---------|------|------|----------|
| **개발 생산성** | Velocity 안정성 | 변동성 < 20% | 스프린트별 |
| **개발 생산성** | 추정 정확도 | 80% 이상 | 스프린트별 |

- 각 지표별 목표 달성 여부
- 미달성 지표 분석
- 개선 계획 수립

## 🔍 지표 리뷰 및 개선

### 리뷰 주기

- **주간**: 주간 지표 리뷰
- **월간**: 월간 지표 리뷰
- **분기별**: 종합 지표 리뷰 및 개선 계획 수립

### 리뷰 항목

   - 개선/악화 추세 확인

3. **개선 계획**
   - 지표 개선 계획 수립
   - 우선순위 결정
