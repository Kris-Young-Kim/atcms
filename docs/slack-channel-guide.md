## 📋 개요

이 문서는 AT-CMP 프로젝트의 Slack 워크스페이스 채널 구성 및 사용 가이드를 설명합니다. 효율적인 팀 커뮤니케이션을 위한 채널 구조를 제시합니다.

---

## 🎯 채널 구성 목표

1. **정보 분류 및 구조화**
2. **효율적인 알림 관리**
3. **프로젝트별 커뮤니케이션 분리**
4. **빠른 정보 검색**

---

## 📊 채널 구조

### 필수 채널

#### 1. `#general` (일반)

**목적**: 프로젝트 전체 공지 및 일반적인 소통

**사용 규칙**:
- 프로젝트 전체 공지사항
- 팀 소개 및 환영 메시지
- 일반적인 질문 및 답변

**알림 설정**: 기본 알림 (모든 메시지)

---

#### 2. `#alerts` (알림)

**목적**: 시스템 알림 및 에러 알림

**사용 규칙**:
- Sentry 에러 알림
- 배포 실패 알림
- 시스템 장애 알림
- 보안 알림

**알림 설정**: 모든 메시지 알림 (긴급)

**통합**:
- Sentry → Slack 알림
- GitHub Actions → Slack 알림
- Vercel → Slack 알림 (선택사항)

---

#### 3. `#performance` (성능)

**목적**: 성능 관련 알림 및 논의

**사용 규칙**:
- 성능 저하 알림
- 느린 API 경고
- Core Web Vitals 알림
- 성능 최적화 논의

**알림 설정**: 중요 메시지만 알림

**통합**:
- Sentry Performance → Slack 알림
- Lighthouse CI → Slack 알림

---

#### 4. `#deployments` (배포)

**목적**: 배포 관련 알림 및 논의

**사용 규칙**:
- 배포 성공/실패 알림
- 배포 일정 공유
- 배포 관련 질문 및 답변
- 롤백 알림

**알림 설정**: 중요 메시지만 알림

**통합**:
- GitHub Actions → Slack 알림
- Vercel → Slack 알림

---

#### 5. `#random` (잡담)

**목적**: 비업무 관련 대화

**사용 규칙**:
- 팀 빌딩
- 일상 대화
- 공유하고 싶은 내용

**알림 설정**: 언급 시에만 알림

## 🔔 알림 설정 가이드

### 알림 레벨

1. **모든 메시지 알림**: 긴급 채널 (`#alerts`)
2. **중요 메시지만 알림**: 일반 채널 (`#general`, `#performance`, `#deployments`)
3. **언급 시에만 알림**: 비업무 채널 (`#random`)

### 언급 규칙

- **@channel**: 모든 팀원에게 알림 (긴급 공지에만 사용)
- **@here**: 현재 온라인인 팀원에게 알림
- **@username**: 특정 사용자에게 알림

---

## 📝 채널 사용 가이드

### 메시지 작성 규칙

1. **명확한 제목**: Thread를 사용하여 토픽별로 정리
2. **컨텍스트 제공**: 링크, 스크린샷, 로그 포함
3. **액션 아이템 명시**: 필요한 경우 명확히 표시

### Thread 사용

- 관련 대화는 Thread로 진행
- 메인 채널은 깔끔하게 유지
- 검색 효율성 향상

### 파일 공유

- 중요 문서는 고정(Pin) 사용
- 파일은 Thread에 첨부 권장
- 링크 대신 파일 직접 첨부 (장기 보관)

## 🔗 외부 도구 통합

### Sentry 통합

**설정 방법**:
1. Sentry Dashboard → Settings → Integrations
2. Slack 통합 선택
3. 워크스페이스 연결
4. 알림 채널 설정:
   - 에러 알림: `#alerts`
   - 성능 알림: `#performance`

### GitHub Actions 통합

**설정 방법**:
1. GitHub 저장소 → Settings → Secrets
2. Slack Webhook URL 추가
3. 워크플로우에서 Slack 알림 설정

**알림 예시**:
- 배포 성공/실패 → `#deployments`
- CI/CD 실패 → `#alerts`
- 성능 테스트 결과 → `#performance`

### Vercel 통합 (선택사항)

**설정 방법**:
1. Vercel Dashboard → 프로젝트 → Settings → Notifications
2. Slack 통합 선택
3. 알림 채널 설정
